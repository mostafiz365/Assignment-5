1️. What is the difference between var, let, and const?
Answer: 
Var: var is the older way of declaring variables. It is function-scoped and can be redeclared and updated, which sometimes causes unexpected problems in code.Present days we should not use it.

Let: let is a modern alternative to var. It is block-scoped, meaning it only works inside the block where it is declared. Its value can be updated.

Const: const is also block-scoped, but its value cannot be changed after it is declared. It used for values that should remain constant.


2️. What is the spread operator (...)?
Answer:
The spread operator (...) is used to expand the elements of an array. It allows values to be copied or combined easily without modifying the original data.


3️. What is the difference between map(), filter(), and forEach()?
Answer:
map() is used to each element of an array and creates a new array after transforming the values.
filter() is also creates a new array, but it only includes the elements that meet a specific condition.
forEach() is used to  simply loops through each element of the array, but it cannot provide a new array or not return an array.


4️. What is an arrow function?
Answer:
An arrow function is a shorter and simpler way to write a function in JavaScript. It was introduced in ES6 to make the code cleaner and easier to read.
Instead of using the traditional function keyword, an arrow function uses the arrow symbol (=>).


5️. What are template literals?
Answer:
Template literals are a modern way of writing strings in JavaScript using backticks.Template literals are especially useful when creating dynamic html and string.
